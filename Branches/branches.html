Branches

São "SEPARAÇÕES" feitas no trabalho de um projeto, seja de um software ou outro, e tem por funcionalidade facilitar o desenvolvimento do projeto
sem precisar tirar ele do ar, tendo como objeto de ferramenta a programação em paralelo, facilitação de desenvolvimento enquanto o site estavél.

  • Novos mecanismos;
  • Ajustes de erros;
  • Mudança de estrutura;

Branch Main

É como se fosse o braço principal do projeto, como se fosse o tronco de uma arvore, é a parte onde será visto pelas pessoas que acessarem o site que
você está desenvolvendo.

Branch Secundária

É a parte onde será feita as atualizações do projeto, é aonde você desenvolvedor vai conseguir inserir coisas novas no seu modelo de site ou excluir,
sem que essas ações sejam "jogadas" diretamente em sua MAIN PRINCIPAL.

Hospedagem

O site que nós utilizamos é o GitHub, pois ele oferece uma hospedagem gratúita de sites, basta você programador criar uma conta que logo logo vai
estar aproveitando tudo o que o GitHub tem a disposição.
