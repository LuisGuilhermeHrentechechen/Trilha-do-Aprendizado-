table

Ela ter por função desenvolver as tabelas, sendo compostas por (table, tr, th e por ultimo a tag td). Quando for utilizada o programador ira colocar primeiro a tag table, após isso ele vai definir a quantidade de linhas da tabela, que é definido pela tag tr, ele pode colocar títulos na tabela, isso com a tag th e por ultimo será utilizado a tag td, que serve para preencher o conteúdo que vai ser inserido dentro das linhas e colunas da tabela.

Del

A tag del serve para riscar o texto, Exemplo: Tag del serve para riscar o testo texto...

Ins

A tag ins serve para sublinhar o texto, Exemplo: Tas ins serve para sublinhar texto...

Sub

A tag sub serve para diminuir o texto e deixar ele mais para baixo, veja o Exemplo: Irati1960

Sup

A tag sup serve para diminuir o texto e deixar ele mais para cima, veja o Exemplo: Irati 2021

Blockquote

O blockquote serve para citações longas, quando você usa informação de outro site ou faz uma grande citação você pode usar a tag blockquote. Veja um exemplo a seguir.

"O crescimento pessoal e profissional está fora da zona de conforto. Aprenda a apreciar a zona de desconforto, pois é lá que o seu futuro é traçado."

Q

A tag q serve para uma citação curta. Exemplo: Jogue pro alto, se voltar é seu.

Abbr

A tag Abbr é uma tag de abreviação, ela abrevia uma frase. Exemplo: IFPR.

Address

A tag address serve para colocar as informações do proprietário do site, pessoa ou empresa. É uma tag para informação de contato.

sobre_index

Cite

A tag cite é para citar um titulo de uma obra, como musicas,filmes, obras, livros, etc… Exeto o nome de pessoas que não se enquadra no cite. Exemplo: A banda Metallica é muito boa.

Bdo

A tag bdo inverte o sentido do texto, podendo ser da direita para a esquerda (rtl) ou da esquerda para a direita (ltr). Exemplo de rtl "Luís Guilherme". Exmplo de ltr "Luís Guilherme".

Code

A tag code permite você utilizar um texto que é um código, permitindo colocar textos que certamente seriam ser entendidos como comandos. Exemplo: Blockquote.

Pre

A tag pre deixa o texto já pré definido, o mesmo texto que foi inserido dentro das tags será o texto que aparecerá para no site, não tem a necessidade de br por exemplo. Exemplo: 

Irati-PR, 
            2021 .

Kbd

A tag kbd tem por funcionalidade mostrar o texto como a entrada do teclado, se você quer que a pessoa use uma tecla do teclado você utiliza essa tag. Exemplo: a tecla F5 recarrega uma página da web.
