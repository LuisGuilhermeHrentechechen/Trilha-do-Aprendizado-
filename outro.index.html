HTML

É uma Linguagem de Marcação de Hipertextos baseada em tags criadas na década de 1990 por Tim Berners-Lee.

A HTML é utilizada para publicação de conteudos de hipertexto, tais como:
  • texto;
  • imagem;
  • audio;
  • vídeo
Estes elementos podem ser vinculados através de hyperlinks.

TAGS

As tags definem ao navegador a marcação de conteúdo.
As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.

As tags podem possuir atributos, ou seja, são parametros que definem algum tipo de informação extra ao navegador.
